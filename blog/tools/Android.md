# 拯救卡顿的Android机

Android机使用一段时间后就会卡顿，主要分为以下几点原因：
1. 硬件跟不上
2. 应用质量差，国内APP稂莠不齐，管控不严
3. 后台管控及推送不统一
4. “相对论”，不要拿一两千的Android和五六千的iPhone比速度（手动狗头）

所以，我们可以尝试以下几种方式解决问题：
1. 买买买
2. 安装原生或类原生Android系统，使用Google Play（需要科学上网）
3. 安装黑域、冰箱、绿色守护等控制后台进程的应用

买买买大家都会，不差钱就行，今天主要介绍安装类原生Android系统
